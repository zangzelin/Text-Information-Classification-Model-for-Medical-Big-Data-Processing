# Text Information Classification Model for Medical Big Data Processing

#### 项目介绍
用于医疗大数据处理的文本信息分类模型
首先使用分词算法与doc2vec算法计算文本间的相似度，然后使用ＫＮＮ聚类方法对文本进行有效聚类。将医生的医嘱文本信息转换为０－１标签信息，方便预测模型进行进一步处理。
